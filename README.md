# Sip

Sip is a dependency injection library for Swift. The API is heavily inspired by https://github.com/square/Cleanse, with key differences in design and implemention.

## Features

* Unique binding
* Scoped binding
* Tagged binding
* Multibinding: Array and Dictionary
* Subcomponents
* Injection method
